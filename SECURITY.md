# Security Policy

## Supported versions

The current `main` branch is supported. Security fixes are coordinated from a
reviewed report rather than promised on a fixed schedule.

## Reporting a vulnerability

Please use GitHub private vulnerability reporting for this repository. Do not
include exploit details, credentials, tokens, or other sensitive information in
a public issue, pull request, or discussion.

We will acknowledge a valid report, assess impact, and coordinate a fix and
disclosure timeline with the reporter. Please allow time for review before
public disclosure.
