# Stage 4 Prompt: Future Profile Roadmap

Use this prompt after Stage 3 is complete and approved.

## Objective

Document deferred profiles without implementing their profile files, examples,
fixtures, or stack-specific smoke checks.

## Scope

Update `docs/future-profiles.md` for `terraform-module`,
`node-fullstack-k8s`, and `python-vue-fullstack-k8s`. Each section must define
discovery rules, declarative commands, Jira subtask expectations, PR review
budget guidance, fixtures, example scaffold, smoke checks, secret boundaries,
and promotion criteria.

Use fake IDs and names. Do not include credentials, private URLs, cloud state,
production identifiers, automated deployments, or remote mutations.

## Validation

Verify deferred assets remain documented but unimplemented, run available
Markdown and diff checks, prepare concise Stage 4 Jira handoff text without
posting it, propose Stage 5, and wait for approval.
